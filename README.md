# s3://bucket/log/viewer


## Setup

### get code & install dependencies 
```bash
git clone git@github.com:flatanimals/s3-logviewer.git
yarn install
```

### run local
```bash
# start local server
yarn serve
# or to launch in a browser window
yarn serve --open
```

### build for deploy
```bash
yarn build
# copy dist/ folder to your hosting environment
```
